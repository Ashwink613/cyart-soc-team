\# Cyart SOC Internship – Week 2 Submission



\## Overview

This repository contains the Week 2 SOC internship tasks completed as part of the Cyart internship program. The focus of this week was on alert prioritization, incident classification, alert triage, incident response workflows, and end-to-end incident documentation.



The work demonstrates SOC analyst thinking through documentation, structured workflows, and incident handling practices.



---



\## Contents



\- \*\*Alert-Prioritization.md\*\*  

&nbsp; Defines alert severity levels using CVSS, impact, urgency, and business context.



\- \*\*Incident-Classification.md\*\*  

&nbsp; Classifies security incidents using MITRE ATT\&CK techniques and contextual metadata.



\- \*\*Incident-Response.md\*\*  

&nbsp; Documents the incident response lifecycle following NIST guidelines.



\- \*\*Alert-Triage.md\*\*  

&nbsp; Demonstrates alert triage, prioritization, and threat intelligence validation.



\- \*\*Capstone-Incident-Report.pdf\*\*  

&nbsp; Full alert-to-response incident report including executive summary, timeline, response actions, and recommendations.



\- \*\*Stakeholder-Briefing.md\*\*  

&nbsp; Non-technical summary prepared for management and stakeholders.
