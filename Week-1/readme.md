\# Cyart SOC Internship – Week 1 (SOC Simulation)



\## Overview

This folder contains the Week 1 SOC simulation task completed as part of the Cyart internship. The focus of this task was on building a mini SOC lab using Splunk SIEM and understanding core SOC operations.



---



\## Activities Performed

\- Built a Splunk-based SOC lab

\- Centralized Windows Security logs

\- Created detection queries for:

&nbsp; - Brute force login attempts

&nbsp; - Unauthorized service creation

&nbsp; - Suspicious process execution

\- Configured automated alerts

\- Created SOC monitoring dashboards

\- Investigated alerts and documented incidents

\- Mapped detections to MITRE ATT\&CK techniques
